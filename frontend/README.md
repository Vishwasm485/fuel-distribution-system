# Fuel Distribution System

A full-stack Fuel Distribution Management System built using:

- Next.js
- TypeScript
- Tailwind CSS
- Flask
- MySQL
- JWT Authentication

## Features

### Admin

- Manage distributors
- View customers
- View feedback
- Dashboard analytics

### Distributor
- Add fuel prices
- Manage bookings
- Accept / Reject / Deliver orders
- Revenue tracking

### Customer
- Register & login
- View distributors
- Book fuel
- Payment simulation
- Track bookings
- Submit feedback

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS
- Framer Motion

### Backend
- Flask
- SQLAlchemy
- JWT Authentication

### Database
- MySQL
